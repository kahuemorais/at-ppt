# AlphaTrust

Institutional presentation website for AlphaTrust, a trading company led by Rodrigo Figueiredo, two-time world trading champion.

## Stack

- HTML5, CSS3, JavaScript (vanilla)
- No frameworks or build tools required
- Google Fonts (Sora + Manrope)

## Features

- Bilingual support (EN/PT) with language switcher
- Responsive design
- Scroll-based header show/hide
- Commission levels showcase with images

## Structure

```
at-ppt/
├── index.html        # Main HTML file
├── styles.css       # All styles
├── app.js           # Language switching logic
├── img/             # Images and assets
├── alphatrustppt.html # Original presentation file
└── PRODUCT.md        # Product documentation
```

## Development

Simply open `index.html` in a browser. No build step required.

## Deploy

Push to `main` branch and GitHub Pages will auto-deploy.
